## 数式

### ブロック

ガウス積分：
$$\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}$$

オイラーの公式：
$$e^{i\theta} = \cos\theta + i\sin\theta$$

自然対数の定義：
$$\lim_{n \to \infty} \left(1 + \frac{1}{n}\right)^n = e$$

テイラー展開：
$$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!} (x-a)^n$$

### インライン

二次方程式 $ax^2 + bx + c = 0$ ($a \neq 0$) の解は：

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

判別式 $D = b^2 - 4ac$ により：

- $D > 0$ のとき、異なる2つの実数解
- $D = 0$ のとき、重解
- $D < 0$ のとき、共役な2つの複素数解
